wrev2

matrix fliping

Parameters

x :

double matrix

M :

1 for row-wise, 2 for column-wise, 3 for both

Y :

fliping result

Description

wrev2 is a fliping utility function on time domain.

Examples

a=rand(3,3);
Y=wrev2(a,1);
