coifwavf

coiflets scaling filter

Parameters

wname :

wavelet name, 'coif1' to 'coif17'

F :

scaling filter

Description

coifwavf is an utility function for obtaining scaling filter of coiflets wavelet.

Examples

F=coifwavf('coif3');
